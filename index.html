    On Rewind API v5.17.4

    Scroll down for code samples, example requests and responses. Select a language for code samples from the tabs above or the mobile navigation menu.

    On Rewind API documentation based on OpenAPI 3.0. This document provides info on all publicly accessible routes if identified with a bearer token.

    Base URLs:

    • https://api-gateway.onrewind.tv

    • https://staging-api-gateway.onrewind.tv

    • https://dev-api-gateway.onrewind.tv

    Email: API Support

    Authentication

    • HTTP Authentication, scheme: bearer A bearer token called "accessToken" is needed to access resources. By default the token will expire after 24h. The bearer token must be provided on every subsequent requests to the API. More info provided on the route 'POST /auth/login'
